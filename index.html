    How to use the GTB solver

    While playing Guess the Build, hints containing underscores and letters will appear. These hints are used to make the game easier, and help out guessers. Using these hints, a list of all possible combinations can be found. These combinations help out the player even more. To get started, just enter the hint containing underscores into the text box. A list of combinations will appear. Click on a combination to copy it quickly.

    Why cant I see any words?

    Make sure you entered in the right amount of underscores. If results are to broad, you might want to wait for more letters to appear, narrowing down results.
